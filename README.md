## Getting Started
### Clone repo

```

git clone https://github.com/aws-samples/travel-personal-assistant.git
cd travel-personal-assistant

```

### Install the packages

```

npm i

```

### Initiate a cloud sandbox environment

```

npx ampx sandbox

```

### Run the App

```

npm run dev

```
